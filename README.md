# Newsletter app

This aplication is an mvp to newslleter manager, you can create multiple newsletter

## How to run
run  ```docker-compose build```

run  ```docker-compose up```

## Technologies
  - Frontend
    - React.js
    - MUI
  - Backend
    - Express.js
    - Mongoose
    - Node mailer
  - Database
    - mongoDb
  - Arc
    - docker
